# Регламент доступа к базам данных

Заглушка слайса S5 (spec/70). Второй документ коллекции — чтобы инжест собирал чанки из
нескольких файлов, а поиск различал темы.

## Роли и права

Доступ выдаётся по ролям. Роль readonly видит только чтение и не меняет данные. Роль владельца
схемы управляет таблицами. Прямой доступ суперпользователя запрещён вне аварийных окон.

## Выдача доступа

Заявка на доступ согласуется владельцем ресурса. Временный доступ выдаётся на срок задачи и
отзывается автоматически. Постоянный доступ пересматривается раз в квартал.
