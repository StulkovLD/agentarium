# 30 · SDK

```
Статус: draft
Владелец: контракт разъёма — что агент получает от платформы и что обязан платформе
Зависит от: 00, 20
```

## Определение

**Опр.** SDK — библиотека `agentarium`, которой агент подключается к платформе. Розетка: прибор любой, разъём один. SDK не знает про LangChain, GigaChat и предметную область — это мозги, они дело агента.

## Весь агент целиком

```python
from agentarium import Agent, Reply

class ParserAgent(Agent):
    async def handle(self, envelope) -> Reply | None:
        text = envelope.payload["text"]
        parsed = await my_brains(text, self.config)   # мозги любые: LangGraph, голый API, регулярки
        return Reply(type="request.parsed", payload=parsed)
```

Запуск: `python -m agentarium run parser` — рантайм читает конфиг экземпляра, поднимает агента.

## Что SDK делает за агента

| Зона | Что именно |
|---|---|
| Подключение | соединение с шиной, объявление своей очереди, подписка по маршрутам из чертежа |
| Протокол | разбор и валидация входного конверта, сборка исходящего: id, trace_id, causation_id, ts, producer — агент их не трогает |
| Подтверждения | ack после успешного `handle`; неподтверждённый конверт вернётся в очередь |
| Повторы и ошибки | ретраи с паузой, после исчерпания — конверт `task.failed` (политика в `50-observability.md`) |
| Наблюдаемость | спан на каждую обработку, structlog с trace_id — бесплатно для любого агента |
| Здоровье | HTTP `/health` для docker healthcheck |
| Конфиг | `self.config` — блок экземпляра из чертежа (модель, коллекция, промпт-профиль…) |
| Склады | `self.storage.postgres()`, `self.storage.qdrant(collection)` — клиенты с проверкой паспорта (`45-storage.md`) |

## Контракт `handle`

- Вход: валидный конверт. Невалидный до `handle` не доходит — SDK отвергает его сам (fail-fast).
- Выход: `Reply(type, payload)` — SDK завернёт и опубликует; `None` — конец ветки, публиковать нечего.
- Тип агента объявляет манифестом, какие типы он принимает и какие производит:

```python
class ParserAgent(Agent):
    consumes = ["request.new"]
    produces = ["request.parsed"]
```

Манифест — контракт типа: рантайм сверяет маршруты чертежа с манифестами на старте и падает при несовместимости (маршрут ведёт тип не туда, где его умеют принять).

## Жизненный цикл

```
старт → конфиг экземпляра → манифест-проверка → соединение с шиной →
→ петля: конверт → span → handle → publish(Reply) → ack →
→ SIGTERM → дообработать текущий конверт → закрыть соединение → выход
```

Изящное завершение обязательно: конверт, начатый в момент остановки, либо доведён и подтверждён, либо не подтверждён и вернётся в очередь. Потерянных конвертов не существует.

## Чего SDK не делает — по принципам

- Не создаёт LLM-клиентов и не импортирует LLM-фреймворки: передаёт агенту конфиг модели как данные — и всё.
- Не маршрутизирует: не решает, кому уйдёт Reply. Тип вернул агент, дальше — чертёж.
- Не прячет ошибки: ретраи исчерпаны — громкий `task.failed`, не тихий фолбэк.
